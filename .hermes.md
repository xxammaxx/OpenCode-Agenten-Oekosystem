# Hermes Project Handoff

## Purpose

This project bundles portable Hermes handoff assets for the OpenCode bootstrap.

## Instructions

- OpenCode is the primary coding executor.
- Hermes acts as the orchestrator and skill runtime.
- Keep MCP servers disabled until explicitly reviewed.
- Prefer the project-local bundle assets instead of rewriting global Hermes config.

## Bundle

- `project-bootstrap`
- `project-reality-refresh`
- `run-card`
- `mcp-selection`
- `hermes-handoff`
- `worktree-safety`
- `checkpoint-and-rollback`
- `living-truth-mirror`
- `remote-ci-approval-gate`
- `provider-neutral-config`

## Safety

- no secret material
- no automatic remote CI
- no autonomous production writes
- no global config overwrite
